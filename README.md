# secret-tunnel
Secret tunnel! Secret tunnel! Through the mountains secret, secret, secret, secret tunnel!


# Run Dev Env

- Install node16.x and npm (use nvm to manage npm and node version)

```
$ npm i
$ npm run dev
$ open -a "Google Chrome" http://localhost:3000
```
